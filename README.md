Batyr
=======

Batyr is a web application that can be used to improve listening skills. It uses the [Tatoeba](http://tatoeba.org) database of example sentences, and presents the user with audiosamples â€“ one sentence at a time. You can try Batyr online at [lltk.de](http://lltk.de).

The name is inspired by [Batyr](http://en.wikipedia.org/wiki/Batyr_(animal)) the elephant.
